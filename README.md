# Simple door alarm system for Raspberry Pi

Get notified when your door opens.

## Installation

Clone the project

    git clone git@github.com:schurig/door-alarm.git


And then execute:

    $ bower install
    $ npm install


## Usage

Run the server with

    $ node app.js


## What it supports:

[x] email notifications
[x] pushover notifications
[x] playing sounds
[] pushbullet notifications
[] web-interface log
[] web hooks
